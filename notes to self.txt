---Dockerfile Problems---------------------------------------------------------

    There can only be one CMD instruction in a Dockerfile or build stage. Only 
    the last one will have an effect.

    We had:     CMD ["svelte-kit", "build"]   // this creates the build folder
                CMD ["node", "build"]         // this runs the build folder
